astrbot插件,用于给引用的消息贴表情(<del>填满狗群友</del>)

# 使用方法
引用你需要贴表情的消息,回复/贴表情或者(/fill /贴等指令)

可以在配置文件中设置默认贴表情数量,贴表情时间间隔,是否开启管理员优待模式(无法被贴表情,贴其他人表情时默认20)

# 帮助
使用指令:/贴表情帮助 以获取更多提示信息

# 注意

**仅aiocqhttp

** 仅群聊中有效

**单条消息贴表情上限为20

**内置的表情id中存在部分QQ已经弃用的表情,无法正常查看
